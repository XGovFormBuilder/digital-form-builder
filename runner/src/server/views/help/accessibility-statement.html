Accessibility Statement

UKHSA is committed to making this website accessible in accordance with the Public Sector Bodies (Website and Mobile Applications) (No.2) Accessibility Regulations 2018. This service complies with the Web Content Accessibility Guidelines version 2.1 AA standard.

These guidelines are the internationally recognised benchmark for building accessible websites.

This website is currently being tested to assess its accessibility standards. Any limitations, issues or risks raised will be formally assessed and improved to ensure this website meets accessibility standards.

This website was tested on 3rd July 2023 using Wave, and we found a few alerts that are not issues or risks. These alerts will be formally assessed as part of our ongoing assessment review. The alerts are:

  • Text appears to be a heading but is not a heading element

Compatibility with tools

This website is run by UKHSA and built on the open-source XGov Form builder. We want as many people as possible to be able to use this website. As per the accessibility done by FCDO for XGov Form builder platform, you should be able to:

  • change colours, contrast levels and fonts
  • zoom in up to 200% without the text spilling off the screen
  • View the browser in a one column format (1280px browser width and Zoom in to 400%)
  • navigate most of the website using just a keyboard
  • navigate most of the website using speech recognition software
  • listen to most of the website using a screen reader (including the most recent versions of JAWS, NVDA and VoiceOver)

We've also made the website text as simple as possible to understand.

The site is also designed to be compatible with the following, and it will work with any browser as long as it supports ES5:

  • basic operating system screen magnifiers, such as ZoomText 11
  • speech recognition software, such as Dragon Naturally Speaking

The website works entirely without javascript, so it should work on a website without javascript.

What to do if you cannot access parts of this website

If you experience difficulty accessing any part of this website, please contact your local health practitioner or simply complete the feedback page.
